# 📌 API Documentation: Get Product List

## 🔗 Endpoint

GET https://api-open.olsera.co.id/api/open-api/v1/en/product

---

## 📖 Deskripsi

Endpoint ini digunakan untuk mengambil daftar produk dari sistem Olsera.

---

## ⚙️ Request

### Method

GET

### Headers

Authorization: Bearer <access_token>  
Content-Type: application/json

---

## 📥 Request Parameters

Tidak ada parameter khusus yang wajib dikirim pada request ini.

---

## 🧪 Contoh Request (cURL)

```bash
curl -X GET "https://api-open.olsera.co.id/api/open-api/v1/en/product" \
  -H "Authorization: Bearer your_access_token_here" \
  -H "Content-Type: application/json"
  -d "{\"id\": 1}"
```
